## Objetivos com a Disciplina

Nesta disciplina de Programação de Aplicativos, meus objetivos são:

1. Compreender os fundamentos de desenvolvimento de aplicativos para dispositivos móveis e desktop.
2. Aprender a usar frameworks e ferramentas modernas de criação de aplicativos.
3. Desenvolver habilidades de programação estruturada e orientada a objetos aplicadas a apps.
4. Criar interfaces gráficas funcionais e intuitivas, seguindo boas práticas de design.
5. Integrar APIs, bancos de dados e serviços externos aos aplicativos desenvolvidos.
6. Construir um portfólio de projetos práticos demonstrando capacidades técnicas aprendidas.
